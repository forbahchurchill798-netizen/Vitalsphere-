# VitalSphere

A modern, beautiful all-in-one health & fitness mobile web app.

### Features
- Dark modern UI design
- Fitness, Nutrition, Mindfulness, Sleep tracking
- Dashboard with stats
- Responsive mobile-first design

### Tech Stack
- HTML5
- CSS3 (Tailwind-inspired custom styles)
- Vanilla JavaScript
